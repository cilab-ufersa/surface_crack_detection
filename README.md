# Surface Crack Detection


## Prerequisites

What things you need to have to be able to run:

  * Python 3.11 +
  * Pip 3+
  * VirtualEnvWrapper is recommended but not mandatory


## Requirements 

```bash
    $ pip install -r requirements.txt
```
